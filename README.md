**WakalaSmart Management System (WSMS)**

 ## Project Overview
WakalaSmart Management System (WSMS) is a web-based application developed to assist mobile money agents in managing their daily operations efficiently. The system provides a centralized platform for recording transactions, monitoring balances, managing float distribution, and tracking profits across different mobile network operators.

 ## Problem Statement
Mobile money agents often rely on manual record keeping, which can lead to errors, inefficiencies, and difficulties in tracking business performance. WSMS aims to provide a digital solution that improves transaction management, financial monitoring, and operational efficiency.

## Objectives

* To automate daily mobile money agent operations.
* To provide real-time account balance monitoring.
* To simplify float management across multiple networks.
* To track daily profits and capital.
* To improve transaction record keeping and reporting.

## System Architecture

The application follows a client-server architecture:

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
* Local Development Environment: XAMPP
